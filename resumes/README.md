# Resume Files

Keep personal resume files here or point `config.toml` to an absolute path elsewhere.

Git ignores:

- `base_resume.md`
- `*.pdf`
- `*.doc`
- `*.docx`
- `private/`

Configure resume paths in `config.toml`:

```toml
mena_resume_path = "/absolute/path/to/resume-with-photo.pdf"
non_mena_resume_path = "/absolute/path/to/resume-without-photo-np.pdf"
```
